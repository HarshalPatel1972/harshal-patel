## 2024-03-05 - Safe Event Throttling without Caching
 **Learning:** When optimizing high-frequency scroll listeners with `requestAnimationFrame`, do NOT cache DOM nodes retrieved via `document.getElementById` inside a generic `useEffect` closure. If the application lazily renders these sections later (e.g. after a preloader finishes or via dynamic imports), the cached node references remain `null` forever and silently break the layout or scroll spy features. Also, never replace inline styling by writing to `element.setAttribute('style', ...)` - it forcefully overwrites any other existing inline styles managed by React or animation libraries. Always modify `element.style.propertyName` directly.
 **Action:** Query the DOM dynamically inside the `requestAnimationFrame` block to ensure elements exist, and mutate only the specific `.style` properties required.
