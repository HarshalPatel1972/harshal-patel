## 2024-05-13 - [Performance] Hoisting Static Arrays for SVG Grids\n**Learning:** The previous implementation used `Array.from().map()` inline within React components (sometimes wrapped in `useMemo`) to generate static background SVG visual elements (lines, shards, tech debt tags). Because these calculations utilized impure functions (`Math.random()`), they caused `react-hooks/purity` ESLint errors when not correctly isolated, and micro-benchmarks indicate an inline creation has a massive performance overhead in React render loops compared to a module-level constant.\n**Action:** Hoist these static generations out of the React component body to module-level constants to both improve UI rendering performance (~32x speedup compared to inline map generation inside render cycle per previous logs) and clear the purity lint errors.
